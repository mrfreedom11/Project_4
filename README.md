# Bank Account Management System

A simple, interactive command-line banking application built with **Python** and **SQLite3**.

## Features
- Create new user accounts with custom initial balance.
- View all registered accounts with their live balance.
- Deposit money into specific accounts.
- Withdraw money safely.
- Transfer money securely between two accounts.

## How to Run
Run the script using Python:
```bash
python pro_4.py
